Is My Server Blocked?

As of today (1-May-2016), Mojang has introduced code that blocks client access to multiplayer servers that are breaking the EULA to the latest snapshots of the vanilla 1.9 client. As many servers are not yet compatible with 1.9 clients, there is no easy way to test to see if you have been blacklisted.

So I have created this website with the help of @xBytezz to let server owners search their IP hash to check for blacklist status. Just type in your IP and it will search for you.
